# React Quiz App

An interactive quiz application built using **React**, **Vite**, and **React Router**. The application allows users to answer questions, track quiz progress, and view results through a structured component-based architecture.

## Features

- Interactive quiz interface
- Multiple quiz pages
- Dynamic question rendering
- Reusable question components
- Result calculation and display
- Client-side routing
- Centralized quiz state management
- Clean React project structure

## Technologies

- React
- Vite
- React Router
- JavaScript (ES6)
- HTML5
- CSS3

## Installation

Clone the repository:

```bash
git clone https://github.com/Siddhika1-tech/react-quiz-app.git
```

Navigate to the project folder:

```bash
cd react-quiz-app
```

Install dependencies:

```bash
npm install
```

## Run the Project

Start the development server:

```bash
npm run dev
```

Open the local development URL in your browser.

## Application Routes

| Route | Description |
|---|---|
